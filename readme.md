# Qikvoice

Simple program to create invoices

## Getting Started
Download files and put into a directory.

Run main.py

Follow instructions

requires a template docx called "template.docx"
### Prerequisites

Uses jninja and DocxTemplate for docx templates

```
Use pip to install required imports
```
